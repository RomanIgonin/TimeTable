Welcome!

### Installation

Start the server first

```json-server --host (your ip-address) -p 3002 --routes src/core/api/routes.json --watch src/core/api/db.json```

And run TimeTable app on ios simulator

```npm run ios```

Or use your physical device

```npm start```
